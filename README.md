# Task Manager App (React + Node.js)

A full-stack task manager built using React and Node.js.

## Tech Stack
- React (Frontend)
- Node.js + Express (Backend)
- REST API
- Vite
- Git/GitHub

## Features
- View tasks
- Add tasks
- Delete tasks
- Client-server architecture
